It can feel overwhelming to catch up on decades of research as you start a
project — everyone felt that way at some point. Below is how Christina reads the
literature. Keep what works for you and leave the rest, with one exception.

## The one non-negotiable

**Use a reference manager.** It will significantly improve your life. Christina
uses [Zotero](https://www.zotero.org/) because it's free and easy (Mendeley or
OneNote also work). Download the Word plugin too. You can store PDFs with notes,
highlights, and links.

Free versions have storage limits. You can store your Zotero library in cloud
storage — [this UW guide](https://guides.lib.uw.edu/research/zotero/addstorage)
explains the procedure; for Zotero 7, the [Attanger](https://github.com/MuiseDestiny/zotero-attanger)
plugin works well (its README is in Chinese, but the plugin is very usable).

## General tips

- Christina almost exclusively uses [Google Scholar](https://scholar.google.com) for literature reviews (Web of Science and ProQuest also work).
- Once you find a good paper, do three things: read its references, check who has cited it ("Cited by #"), and download the citation into your library (Cite → RefMan for a `.ris` file that double-clicks into Zotero).
- When you can't access a paper, search through the [UC Davis Library](https://www.library.ucdavis.edu/); off campus, use the [library VPN](https://library.ucdavis.edu/vpn/).
- Create a new Zotero collection per project so papers are easy to find again.
- Keep jot notes in the reference manager.

<div class="callout callout--warn" markdown="1">
**Be cautious with AIAA papers.** Conference papers are not peer-reviewed but can
look identical to journal articles — check the header/footer. It's okay (but less
desirable) to cite a conference paper; always check whether a journal version exists.
</div>

## A paper-review process

1. Search keywords on Google Scholar.
2. Read the abstract.
3. Download the `.ris` file into a Zotero collection.
4. Read the conclusion.
5. Add jot notes to the file.
6. Look at the figures and read the captions (*this is why figures matter so much*).
7. If relevant, read the results.
8. If relevant or replicating, read the methods.

## Tools

Citation-graph tools that visualize how papers relate:
[Inciteful](https://inciteful.xyz/), [Connected Papers](https://www.connectedpapers.com/),
and [Citation Gecko](https://citationgecko.azurewebsites.net/).
